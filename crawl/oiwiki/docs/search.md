# 搜索部分简介

搜索，也就是对状态空间进行枚举，通过穷尽所有的可能来找到最优解，或者统计合法解的个数．

搜索有很多优化方式，如减小状态空间，更改搜索顺序，剪枝等．

搜索是一些高级算法的基础．在 OI 中，纯粹的搜索往往也是得到部分分的手段，但可以通过纯粹的搜索拿到满分的题目非常少．
