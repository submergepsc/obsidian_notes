# 数据结构部分简介

数据结构是在计算机中存储、组织数据的方式．小到变量、数组，大到线段树、平衡树，都是数据结构．

程序运行离不开数据结构，不同的数据结构又各有优劣，能够处理的问题各不相同，而根据具体问题选取合适的数据结构，可以大大提升程序的效率．所以，学习各种各样的数据结构是很有必要的．
