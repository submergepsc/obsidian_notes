# 图论部分简介

**图论 (Graph theory)** 是数学的一个分支，图是图论的主要研究对象．**图 (Graph)** 是由若干给定的顶点及连接两顶点的边所构成的图形，这种图形通常用来描述某些事物之间的某种特定关系．顶点用于代表事物，连接两顶点的边则用于表示两个事物间具有这种关系．
