# CCR Plus

## CCR Plus

**CCR Plus** 是一款适用于 NOI 系列比赛的开源的跨平台测评环境，使用 Qt 编写，目前支持 Windows 和 Linux．

源代码托管于 [sxyzccr/CCR-Plus](https://github.com/sxyzccr/CCR-Plus)．
