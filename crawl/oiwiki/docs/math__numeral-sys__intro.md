# 数字系统简介

Warning

此处讨论的不是 [number system](https://en.wikipedia.org/wiki/Number#Classification)，数学中的数字系统（number system）指的是一类集合，如整数集 𝐙Z![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7)、复数集 𝐂C![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7) 等．

**数字系统** （numeral system），又称 **记数系统** ，指的是用以表示数字的书写系统，如 [印度–阿拉伯数字系统](https://en.wikipedia.org/wiki/Hindu%E2%80%93Arabic_numeral_system)、[罗马数字](https://en.wikipedia.org/wiki/Roman_numerals)、[苏州码子](https://en.wikipedia.org/wiki/Suzhou_numerals) 等．数字系统是我们给数做编码的工具．

一般来说，一个数字系统下的数字都是一串符号，同时有一套规则将这串符号和对应的数一一对应起来，例如罗马数字 XLIIXLII![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7)、二进制数 101010(2)101010(2)![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7) 和十进制数 4242![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7) 均能对应到相同的数．

算法竞赛中，我们一般只关注不同数字系统间的转化关系，详见 [进位制](../base/)．
