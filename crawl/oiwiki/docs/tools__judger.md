# 评测工具简介

**评测软件** 是用于本地测试分数的软件．使用者在将代码提交到 OJ 前，可以使用评测软件对自己的程序估分．

本部分介绍了以下几种评测软件：

  * [Arbiter](arbiter/)
  * [CCR Plus](ccr-plus/)
  * [Cena](cena/)
  * [Lemon](lemon/)
