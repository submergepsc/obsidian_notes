# 工具软件简介

本章主要介绍与竞赛有关的工具软件，包括一些代码编辑器的介绍和 OJ 相关工具．

程序是解决 OI 问题的工具，熟练运用代码编辑器是学习 OI 的前提．

了解 OJ 相关的工具能让 OI 之旅更加舒适、便捷．
