# Linux pppsetup命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux pppsetup命令用于设置PPP连线。

这是Slackware发行版内附程序，它具有互动式的问答界面，让用户轻易完成PPP的连线设置。

### 语法

```bash
pppsetup
```


### 实例

设置ppp拨号

```bash
# pppsetup
```
