# Linux logout命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux logout命令用于退出系统。

logout指令让用户退出系统，其功能和login指令相互对应。

### 语法

```bash
logout
```


### 实例

退出系统：

```bash
[root@runoob.com ~]# logout
```
