# Linux logname命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux logname命令用于显示用户名称。

执行logname指令，它会显示目前用户的名称。

### 语法

```bash
logname [--help][--version]
```


**参数** ：

  * \--help 在线帮助。
  * \--vesion 显示版本信息。



### 实例

显示登录账号的信息：

```bash
# logname root
```
