# Linux chfn命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux chfn命令提供使用者更改个人资讯，用于 finger and mail username

使用权限：所有使用者。

### 语法

```bash
shell>> chfn
```


### 实例

改变finger信息

```bash
# chfn Changing finger information for root. Name [root]: hnlinux Office []: hn Office Phone []: 888888 Home Phone []: 9999999 Finger information changed.
```


改变账号真实姓名

```bash
# chfn -f hnunix Changing finger information for root. Finger information changed.
```
