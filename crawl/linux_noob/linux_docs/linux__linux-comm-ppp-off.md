# Linux ppp-off命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux ppp命令用于关闭ppp连线。

这是Slackware发行版内附的程序，让用户切断PPP的网络连线。

### 语法

```bash
ppp-off
```


### 实例

关闭ppp连线

```bash
# ppp-off
```
