# Linux setenv 命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux setenv 命令用于查询或显示环境变量。

setenv 为 tsch 中查询或设置环境变量的指令。

### 语法

```bash
setenv [变量名称][变量值]
```


### 实例

显示环境变量

```bash
setenv
```


设置环境变量

```bash
# setenv USER lx138
```
