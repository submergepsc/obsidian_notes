Java 实例 - 数字求和运算 

以下实例演示了使用do...while结构求0~100的整数数字之和： 

Main.java 文件 

public class Main { public static void main ( String [ ] args ) { int limit = 100 ; int sum = 0 ; int i = 1 ; do { sum = sum + i ; i ++; } while ( i <= limit ) ; System . out . println ( " sum= " + sum ) ; } } 

以上代码运行输出结果为： 

sum=5050
