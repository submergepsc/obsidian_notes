XQuery 教程 

XQuery 之于 XML 作用就类似于 SQL 之于数据库的作用。 

XQuery 被设计用来查询 XML 数据。 

现在开始学习 XQuery ! 

XQuery 实例 

for $x in doc("books.xml")/bookstore/book 
where $x/price>30 
order by $x/title 
return $x/title
