实例 

一个简单的 HTML 表格，包含两列两行： 

<table border="1"> 
<tr> 
<th>Month</th> 
<th>Savings</th> 
</tr> 
<tr> 
<td>January</td> 
<td>$100</td> 
</tr> 
</table>

浏览器支持 

所有主流浏览器都支持 <tr> 标签。 

标签定义及使用说明 

<tr> 标签定义 HTML 表格中的行。 

一个 <tr> 元素包含一个或多个 <th> 或 <td> 元素。 

HTML 4.01 与 HTML5之间的差异 

在 HTML 5 中，不支持 <tr> 标签在 HTML 4.01 中的任何属性。
