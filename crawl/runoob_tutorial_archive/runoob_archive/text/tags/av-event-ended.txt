定义和用法 

ended 事件在音频/视频(audio/video)播放完成后触发。 

该事件通常用于提示播放完成的信息，例如 "谢谢收听", 
"谢谢观看" 等。

语法 

HTML 中：

JavaScript 中：

JavaScript 中, 使用 addEventListener() 方法:

注意： Internet Explorer 8 及更早 IE 版本的浏览器不支持 addEventListener() 方法。
