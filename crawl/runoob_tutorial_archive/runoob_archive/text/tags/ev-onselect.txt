浏览器支持 

所有主流浏览器都支持 onselect 事件属性 

定义和用法 

onselect 属性在选取元素文本后触发。 

onselect 属性可使用于以下元素: <input type="file">, <input type="password">, <input type="text">, <keygen>, 
和 <textarea>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onselect=" script "> 

属性值 

值 

描述 

script 

规定该onselect事件触发时执行的脚本。
