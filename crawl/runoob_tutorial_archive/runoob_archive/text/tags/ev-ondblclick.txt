浏览器支持 

所有主流浏览器都支持 ondblclick 事件属性 

定义和用法 

ondblclick 属性在元素上双击鼠标时触发。 

注意： ondblclick 属性不能使用于以下函数： <base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element ondblclick=" script "> 

属性值 

值 

描述 

script 

规定该ondblclick事件触发时执行的脚本。
