实例

浏览器支持 

所有主流浏览器都支持 onmouseover 事件属性 

定义和用法 

onmouseover 属性在鼠标指针移至元素之上时触发。 

注意： onmouseover 属性不能使用于以下元素：<base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onmouseover=" script "> 

属性值 

值 

描述 

script 

规定该onmouseover事件触发时执行的脚本。
