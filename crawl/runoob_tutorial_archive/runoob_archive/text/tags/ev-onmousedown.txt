HTML onmousedown 事件属性 

实例 

当按下鼠标按钮时运行脚本: 

<p onmousedown="mouseDown()">Click the text!</p> 

尝试一下 » 

浏览器支持 

所有主流浏览器都支持 onmousedown 事件属性 

定义和用法 

onmousedown 属性在按下鼠标按钮时触发。 

提示： 与 onmousedown 事件相关的事件触发顺序 (左/中鼠标按钮): 

onmousedown 

onmouseup 

onclick 

与 onmousedown 事件相关的事件触发顺序(右边鼠标按钮): 

onmousedown 

onmouseup 

oncontextmenu 

注意： onmousedown 属性不能使用于以下元素: <base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onmousedown=" script "> 

属性值 

值 

描述 

script 

规定该onmousedown事件触发时执行的脚本。
