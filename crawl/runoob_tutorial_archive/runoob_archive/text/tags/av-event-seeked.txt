定义和用法 

seeked 事件在用户已移动/跳跃到音频/视频（audio/video）中的新位置时触发。 

提示： seeked 事件的相反事件为 seeking 事件。 

提示： 使用 Audio/Video 对象的 currentTime 属性来获取视频的播放位置。

// 获取 id="myVideo" 的 <video> 元素 
var vid = 
 document.getElementById("myVideo"); 

// 为 <video> 添加 seeked 事件, 在寻址操作完成后执行函数 
vid.addEventListener("seeked", myFunction); 

function myFunction() { 
// 显示在 id="demo" p 元素中 <video> 的当前播放位置 
document.getElementById("demo").innerHTML = vid.currentTime; 
}
