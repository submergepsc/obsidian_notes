定义和用法 

play 事件在音频/视频(audio/video)开始播放时触发。 

提示： pause 事件在音频/视频(audio/video) 暂停播放时触发。

语法 

HTML 中： 

< audio|video onplay=" myScript "> 尝试一下 

JavaScript 中： 

audio|video .onplay=function(){ myScript }; 尝试一下 

JavaScript 中, 使用 addEventListener() 方法: 

audio|video .addEventListener("play", myScript ); 尝试一下 

注意： Internet Explorer 8 及更早 IE 版本不支持 addEventListener() 方法。 

技术细节

Audio, Video
