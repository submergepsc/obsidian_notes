浏览器支持 

所有主流浏览器都支持 lang 属性 

定义和用法 

lang 属性规定元素内容的语言。 

HTML 4.01 与 HTML5之间的差异 

在 HTML5 中, lang 属性可用于任何的 HTML 元素 (它会验证任何HTML元素。但不一定是有用)。 

在 HTML 4.01 中, lang 属性不能用于: 
<base>, <br>, <frame>, <frameset>, <hr>, <iframe>, <param>, 和 <script>。 

语法 

< element lang=" language_code ">
