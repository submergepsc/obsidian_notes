浏览器支持 

所有主流浏览器都支持 onclick 事件属性 

定义和用法 

onclick 属性在单击鼠标时触发。 

注意： onclick 属性不能应用于以下元素：<base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onclick=" script ">

值 

描述 

script 

规定该onclick事件触发时执行的脚本。
