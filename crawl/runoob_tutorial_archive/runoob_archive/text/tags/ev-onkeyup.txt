浏览器支持 

所有主流浏览器都支持 onkeyup 事件属性 

定义和用法 

onkeyup 属性在用户松开按键（在键盘上）时触发。 

提示： 与 onkeyup 事件相关的事件触发顺序: 

onkeydown 

onkeypress 

onkeyup 

注意： onkeyup 属性不能使用于以下元素：<base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或 <title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onkeyup=" script "> 

属性值 

值 

描述 

script 

规定该onkeyup事件触发时执行的脚本。
