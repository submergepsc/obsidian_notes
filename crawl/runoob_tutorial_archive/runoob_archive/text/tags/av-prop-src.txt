HTML 音频/视频 DOM src 属性 

实例 

改变视频的来源： 

myVid=document.getElementById("video1"); 
myVid.src="movie.ogg"; 

尝试一下 » 

定义和用法 

src 属性设置或返回音频/视频的当前来源。 

浏览器支持 

Internet Explorer 10、Firefox、Opera、Chrome 和 Safari 支持 src 属性。 

注意： Internet Explorer 9 及之前的版本不支持 src 属性。 

语法 

设置 src 属性： 

audio|video .src= URL 

返回 src 属性： 

audio|video .src 

属性值 

值 

描述 

URL 

规定音频/视频来源的 URL。 

可能的值： 

绝对 URL - 指向另一个网站（比如 src="http://www.example.com/movie.ogg"） 

相对 URL - 指向网站内的一个文件（比如 src="movie.ogg） 

技术细节 

返回值： 

字符串值，当前的音频/视频来源。
