React componentDidUpdate() 方法 

componentDidUpdate() 方法格式如下： 

componentDidUpdate(prevProps, prevState, snapshot) 

componentDidUpdate() 方法在组建更新后会被立即调用。 

首次渲染不会执行此方法。 

你也可以在 componentDidUpdate() 中直接调用 setState()，但请注意它必须被包裹在一个条件语句里。
