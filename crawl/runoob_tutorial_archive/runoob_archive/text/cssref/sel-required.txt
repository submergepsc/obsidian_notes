CSS :required 选择器 

实例 

如果 input 元素设置了 "required" 属性，设置其为黄色: 

input:required 
{ 
background-color: yellow; 
}

定义和使用 

:required 选择器在表单元素是必填项时设置指定样式。 

表单元素可以使用 required 属性来设置必填项。 

注意： :required 选择器只适用于表单元素: input, select 和 textarea。
