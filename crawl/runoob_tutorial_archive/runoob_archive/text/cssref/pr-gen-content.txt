CSS content 属性 

实例 

以下实例将在每个链接后的括号内加上网址（href 属性）： 

a :after { content: " (" attr ( href ) ")" ; }

属性定义及使用说明 

content 属性与 :before 及 :after 伪元素配合使用，来插入内容。 

默认值： 

normal 

继承： 

no 

版本： 

CSS2 

JavaScript 语法： 

object .style.content="url(beep.wav)"

content 语法与属性值 

语法格式： 

content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit; 

值 

说明 

实例 

none 

设置 content 为空值。

normal 

在 :before 和 :after 伪类元素中会被视为 none，即也是空值。

counter 

设定计数器，格式可以是 counter(name) 或 counter(name,style) 。产生的内容是该伪类元素指定名称的最小范围的计数；格式由style指定（默认是'decimal'——十进制数字）
