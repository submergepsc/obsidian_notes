实例

p 
{ 
border-style:solid;

} 

尝试一下 » 

属性定义及使用说明 

border-top-width属性设置一个元素的顶部边框的宽度。 

注意: 请始终在 border-top-width 属性之前声明 border-style 属性。元素只有在获得边框之后，才能改变其边框的宽度。 

默认值: 

medium 

继承: 

no 

版本: 

CSS1 

JavaScript 语法: 

object object.style.borderTopWidth="thick"
