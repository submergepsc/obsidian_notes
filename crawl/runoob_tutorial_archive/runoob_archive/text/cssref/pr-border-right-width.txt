CSS border-right-width 属性 

实例 

设置右边框的宽度： 

p { border-style: solid ; border-right-width: 15 px ; } 

尝试一下 » 

属性定义及使用说明 

border-right-width属性设置一个元素右边框的宽度。 

注意 请始终在 border-right-width 属性之前声明 border-style 属性。元素只有在获得边框之后，才能改变其边框的宽度。 

默认值： 

medium 

继承性： 

no 

版本： 

CSS1 

JavaScript 语法： 

object .style.borderRightWidth="thick"
