CSS max-height 属性 

实例 

设置段落的最大高度： 

p { max-height: 50 px ; } 

尝试一下 » 

属性定义及使用说明 

max-height 属性设置元素的最大高度。 

注意： max-height属性不包括填充，边框，或页边距！ 

默认值： 

none 

继承： 

no 

版本： 

CSS2 

JavaScript 语法： 

object .style.maxHeight="10px"

属性值 

值 

描述 

none 

默认。定义对元素被允许的最大高度没有限制。 

length 

定义元素的最大高度值。 

% 

定义基于包含它的块级对象的百分比最大高度。 

inherit 

规定应该从父元素继承 max-height 属性的值。
