标签定义及使用说明 

border 是边框属性的简写属性。 

CSS border 属性用于指定元素边框的样式、宽度和颜色。 

可以设置的属性分别（按顺序）：border-width, border-style 和 border-color。 

也可以只设置一个值，例如 border：＃FF0000; 是正确的，其他值会设置成对应属性的初始值。 

语法格式 

可以使用下面列出的一个，两个或三个值来指定 border 属性，值的顺序无关紧要： 

/* 边框样式 */ 

border : solid ; 

/* 边框宽度 | 边框样式 */ 

border : 2px dotted ; 

/* 边框样式 | 边框颜色 */ 

border : outset #f33 ; 

/* 边框宽度 | 边框样式 | 边框颜色 */ 

border : medium dashed green ; 

/* 全局值 */ 

border : inherit ; 

border : initial ; 

border : unset ; 

注意： 如果边框的样式未定义，它将不可见。这是因为样式默认 为none。
