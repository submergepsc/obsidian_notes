CSS z-index 属性 

实例 

设置图像的 z-index： 

img { position: absolute ; left: 0 px ; top: 0 px ; z-index: - 1 ; } 

尝试一下 » 

属性定义及使用说明 

z-index 属性指定一个元素的堆叠顺序。 

拥有更高堆叠顺序的元素总是会处于堆叠顺序较低的元素的前面。 

注意： z-index 进行定位元素(position:absolute, position:relative, or position:fixed)。 

默认值： 

auto 

继承： 

no 

版本： 

CSS2 

JavaScript 语法： 

object .style.zIndex="1"
