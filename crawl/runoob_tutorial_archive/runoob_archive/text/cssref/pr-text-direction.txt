CSS direction 属性 

实例 

设置文字方向"right-to-left": 

div { direction: rtl ; unicode-bidi: bidi-override ; } 

尝试一下 » 

属性定义及使用说明 

direction属性指定文本方向/书写方向。 

默认值： 

ltr 

继承： 

yes 

版本： 

CSS2 

JavaScript 语法： 

object .style.direction="rtl" 

浏览器支持 

表格中的数字表示支持该属性的第一个浏览器版本号。 

紧跟在 -webkit-, -ms- 或 -moz- 前的数字为支持该前缀属性的第一个浏览器版本号。

属性值 

值 

描述 

ltr 

默认。文本方向从左到右。 

rtl 

文本方向从右到左。 

inherit 

规定应该从父元素继承 direction 属性的值。
