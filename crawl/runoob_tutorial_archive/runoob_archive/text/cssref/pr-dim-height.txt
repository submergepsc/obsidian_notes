属性定义及使用说明 

height属性设置元素的高度。 

注意： height属性不包括填充，边框，或页边距！ 

默认值： 

auto 

继承： 

no 

版本： 

CSS1 

JavaScript 语法： 

object .style.height="50px"

属性值 

值 

描述 

auto 

默认。浏览器会计算出实际的高度。 

length 

使用 px、cm 等单位定义高度。 

% 

基于包含它的块级对象的百分比高度。 

inherit 

规定应该从父元素继承 height 属性的值。
