属性定义及使用说明 

float属性指定一个盒子（元素）是否应该浮动。 

注意： 绝对定位的元素忽略float属性！ 

默认值： 

none 

继承： 

no 

版本： 

CSS1 

JavaScript 语法： 

object .style.cssFloat="left"

属性值 

值 

描述 

left 

元素向左浮动。 

right 

元素向右浮动。 

none 

默认值。元素不浮动，并会显示在其在文本中出现的位置。 

inherit 

规定应该从父元素继承 float 属性的值。
