CSS :first-line 选择器 

实例 

每个<p>元素的第一行选择的样式： 

p:first-line 
{ 
background-color:yellow; 
} 

尝试一下 » 

定义和用法 

:first-line选择器用来指定选择器第一行的样式。 

注意: :first-line选择器可以使用以下属性： 

font properties 

color properties 

background properties 

word-spacing 

letter-spacing 

text-decoration 

vertical-align 

text-transform 

line-height 

clear 

注意: "first-line" 选择器适用于块级元素中。
