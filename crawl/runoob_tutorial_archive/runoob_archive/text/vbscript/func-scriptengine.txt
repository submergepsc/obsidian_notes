VBScript ScriptEngine、ScriptEngineBuildVersion、ScriptEngineMajorVersion、ScriptEngineMinorVersion 函数 

ScriptEngine 函数 

ScriptEngine 函数返回当前使用的脚本语言。 

该函数可返回下面的字符串： 

VBScript - 指示当前使用的脚本引擎是 Microsoft Visual Basic Scripting Edition 

JScript - 指示当前使用的编写脚本引擎是 Microsoft JScript 

VBA - 指示当前使用的编写脚本引擎是 Microsoft Visual Basic for Applications 

ScriptEngineBuildVersion 函数 

ScriptEngineBuildVersion 函数返回当前使用的脚本引擎的内部版本号。 

ScriptEngineMajorVersion 函数 

ScriptEngineMajorVersion 函数返回当前使用的脚本引擎的主版本号。 

ScriptEngineMinorVersion 函数 

ScriptEngineMinorVersion 函数返回当前使用的脚本引擎的副版本号。 

语法 

ScriptEngine 

ScriptEngineBuildVersion 

ScriptEngineMajorVersion 

ScriptEngineMinorVersion

document.write(ScriptEngineMinorVersion) 

</script>
