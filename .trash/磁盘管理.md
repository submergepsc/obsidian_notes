在 Linux (Ubuntu) 终端中查看磁盘占用情况，有几个极其好用的命令和工具。既然你习惯使用终端，我为你推荐从“快速总览”到“深度分析”的三种主流方法：

### 方法一：最快的大局观（看各个盘的剩余空间）
打开终端，输入以下命令：

```bash
df -h
```

* **`df`** 代表 Disk Free。
* **`-h`** 代表 Human-readable（将字节换算成你能看懂的 GB、MB 格式）。

**怎么看结果：**
它会列出你所有的硬盘分区（包括你刚才挂载的 Windows D盘）。主要看这几列：
* `Size`：总容量
* `Used`：已用空间
* `Avail`：可用空间
* `Use%`：使用率百分比
* `Mounted on`：挂载点（比如 `/` 是你的 Linux 系统盘，`/media/loviya/...` 是你的 Windows 盘）。

---

### 方法二：极客专属的交互式神器（强烈推荐）
如果你想知道**到底是哪个文件夹把空间吃光了**，强烈推荐安装并使用 `ncdu`。它是终端下最好用的磁盘分析工具，可以直接在终端里像文件管理器一样上下移动并删除大文件。

**1. 安装它：**
```bash
sudo apt install ncdu
```

**2. 运行它（分析你的主目录）：**
```bash
ncdu ~
```
*(如果想分析整个电脑的所有磁盘，可以输入 `sudo ncdu /`，但由于文件太多，扫描可能需要一两分钟。)*

**怎么用：**
扫描完成后，它会按从大到小的顺序排列当前目录下的所有文件夹。你可以：
* 用键盘 **上下方向键** 选择文件夹。
* 按 **回车键** 进入该文件夹，继续看里面的大小分布。
* 按 **左方向键** 返回上一级。
* 如果确定某个文件/文件夹是垃圾，选中它按 **`d` 键** 可以直接删除（注意：删除不可逆！）。
* 按 **`q` 键** 退出。

---

### 方法三：图形界面（如果你不想敲代码）
Ubuntu 系统自带了一个非常直观的图形化磁盘分析工具。

1. 按 `Win` 键打开应用搜索，输入 **“磁盘使用情况分析器”** 或英文 **“Disk Usage Analyzer”**（它的真名叫 Baobab）。
2. 打开后，它会用圆环图非常漂亮地展示各个目录的占比结构，哪里红了一点就钻进去看，非常直观。

### 总结建议
* 只是想看硬盘还剩多少 GB：敲 `df -h`。
* 想清理垃圾或者找大文件：敲 `ncdu`。
