# Linux gzexe命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux gzexe命令用于压缩执行文件。

gzexe是用来压缩执行文件的程序。当您去执行被压缩过的执行文件时，该文件会自动解压然后继续执行，和使用一般的执行文件相同。

### 语法

```bash
gzexe [-d][执行文件...]
```


**参数** ：

  * -d 解开压缩文件。



### 实例

压缩可执行文件

```bash
# gzexe abc
```
