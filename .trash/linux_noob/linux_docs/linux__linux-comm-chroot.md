# Linux chroot 命令

[![Linux 命令大全](./images/up.gif) Linux 命令大全](linux-command-manual.html)

Linux chroot (英文全称：change root) 命令用于改变根目录。

chroot 命令把根目录换成指定的目的目录。

### 语法

```bash
chroot [--help][--version][目的目录][执行指令...]
```


**参数说明** ：

  * \--help 在线帮助。
  * \--version 显示版本信息。



### 实例

改变根目录

```bash
# chroot /mnt/ls //改变根目录
```
