VBScript Array 函数 

Array 函数返回一个包含数组的变量。 

注意： 数组中的第一个元素是零。 

语法 

Array(arglist) 

参数 

描述 

arglist 

必需。数组中元素值的列表（由逗号分割）。 

实例 

实例 1 

<script type="text/vbscript"> 

a=Array(5,10,15,20) 
document.write(a(3)) 

</script> 

以上实例输出结果： 

20
