SQL SELECT 语句 

SELECT 语句用于从数据库中选取数据。 

结果被存储在一个结果表中，称为结果集。 

SQL SELECT 语法 

SELECT column1, column2, ...
FROM table_name; 

与 

SELECT * FROM table_name; 

参数说明： 

column1, column2, ... ：要选择的字段名称，可以为多个字段。如果不指定字段名称，则会选择所有字段。 

table_name ：要查询的表名称。 

* : 通配符，表示选择表中的所有列。
