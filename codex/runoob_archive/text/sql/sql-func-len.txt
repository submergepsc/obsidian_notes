LEN() 函数 

LEN() 函数返回文本字段中值的长度。 

SQL LEN() 语法 

SELECT LEN(column_name) FROM table_name; 

MySQL 中函数为 LENGTH(): 

SELECT LENGTH(column_name) FROM table_name; 

演示数据库 

在本教程中，我们将使用 RUNOOB 样本数据库。

SQL LEN() 实例 

下面的 SQL 语句从 "Websites" 表中选取 "name" 和 "url" 列中值的长度： 

实例 

SELECT name, LENGTH(url) as LengthOfURL 
FROM Websites;
