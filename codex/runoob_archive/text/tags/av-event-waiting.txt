HTML Audio/Video DOM waiting 事件 

实例 

当视频开始播放前需要缓冲下一帧时弹出提示信息： 

var vid = document.getElementById("myVideo"); 
vid.onwaiting = function() 
 { 
alert("Wait! I need to buffer the next frame"); 
}; 

尝试一下 » 

定义和用法 

waiting 事件在视频由于需要缓冲下一帧而停止时触发。 

该事件可用于 <audio> 元素，但通常都用在视频中。

语法 

HTML 中： 

< audio|video onwaiting=" myScript "> 尝试一下 

JavaScript 中： 

audio|video .onwaiting=function(){ myScript }; 尝试一下 

JavaScript 中, 使用 addEventListener() 方法: 

audio|video .addEventListener("waiting", myScript ); 尝试一下 

注意： Internet Explorer 8 及更早 IE 版本不支持 addEventListener() 方法。
