浏览器支持 

目前主流浏览器并不支持 <menu> 标签。 

标签定义及使用说明 

<menu> 标签定义了一个命令列表或菜单。 

<menu> 标签通常用于文本菜单，工具条和命令列表选项。 

提示和注释 

提示: 使用 CSS 来定义菜单列表样式。 

HTML 4.01 与 HTML5之间的差异 

HTML 4.01的 <menu> 元素已废弃。 

HTML5 中 <menu> 元素已被重新定义。
