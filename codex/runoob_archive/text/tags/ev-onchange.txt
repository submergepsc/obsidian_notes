实例

浏览器支持 

所有主流浏览器都支持 onchange 事件属性 

定义和用法 

onchange 属性在元素值改变时触发。 

onchange 属性可适用于 <input>, <textarea>, 和 <select> 元素。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onchange=" script "> 

属性值 

值 

描述 

script 

规定该onchange事件触发时执行的脚本。
