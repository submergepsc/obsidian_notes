浏览器支持 

所有主流浏览器都支持 onblur 事件属性 

定义和用法 

onblur 在元素失去焦点时触发。 

Onblur 通常用于代码验证(当用户离开表单输入域时)。 

提示： onblur 属性的相反属性 onfocus 属性。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onblur=" script "> 

属性值 

值 

描述 

script 

规定该onblur事件触发时执行的脚本
