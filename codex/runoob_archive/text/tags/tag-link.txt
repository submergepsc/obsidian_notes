浏览器支持 

所有主流浏览器都支持 <link> 标签。 

标签定义及使用说明 

<link> 标签定义文档与外部资源的关系。 

<link> 标签最常见的用途是链接样式表。 

注意： link 元素是空元素，它仅包含属性。 

注意： 此元素只能存在于 head 部分，不过它可出现任何次数。 

HTML 4.01 与 HTML5之间的差异 

一些 HTML 4.01 属性在 HTML5 中不支持。 

HTML5 新增了 "sizes" 属性。 

HTML 与 XHTML 之间的差异 

在 HTML 中，<link> 标签没有结束标签。 

在 XHTML 中，<link> 标签必须被正确地关闭。
