浏览器支持 

目前大多数浏览器支持 <legend> 标签。 

标签定义及使用说明 

The <legend> 元素为 <fieldset> 元素定义标题。 

HTML 4.01 与 HTML5之间的差异 

在 HTML 4.01中 "align" 属性已被废弃, HTML5不支持该属性。不建议使用。 
请使用 CSS 来设置 <legend> 元素的对齐方式。
