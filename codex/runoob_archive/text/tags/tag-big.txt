HTML <big> 标签 - HTML5 不支持 

实例 

让文本比常规的字体大一号： 

< p > < big > 这个文本比较大。 </ big > </ p > 

尝试一下 » 

浏览器支持 

所有主流浏览器都支持 <big> 标签。 

标签定义及使用说明 

HTML5 不支持 <big> 标签。请用 CSS 代替。 

<big> 标签用来制作更大的文本。 

提示和注释 

提示： 在文档中使用 CSS 来规定 文本大小 。 

HTML 4.01 与 HTML5之间的差异 

HTML5 不支持 <big> 标签，HTML 4.01 支持 <big> 标签。 

标准属性 

在 HTML 4.01 中，<big> 标签支持如下标准属性：

值
