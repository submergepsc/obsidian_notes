浏览器支持 

所有主流浏览器都支持 onkeypress 事件属性 

定义和用法 

onkeypress 属性在按下按键时触发。 

提示： 与 onkeypress 事件相关的事件触发顺序: 

onkeydown 

onkeypress 

onkeyup 

注意： onkeypress 事件在所有浏览器中不能触发所有按键(例如：ALT, CTRL, 
SHIFT, ESC) 。如果只对用户是否已经按下一个按键检测，
可以使用 onkeydown 取代, onkeydown被所有按键触发。 

注意： onkeypress 属性不能使用与以下元素: <base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>。 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onkeypress=" script ">

值

script
