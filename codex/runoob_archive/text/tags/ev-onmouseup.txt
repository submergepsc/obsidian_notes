HTML onmouseup 事件属性 

实例 

当松开鼠标按钮时运行脚本： 

<p onmouseup="mouseUp()">Click the text!</p> 

尝试一下 » 

浏览器支持 

所有主流浏览器都支持 onmouseup 事件属性 

定义和用法 

onmouseup 属性在用户松开鼠标按键时触发。 

提示： 与 onmouseup 事件相关的事件触发顺序 (左/中鼠标按钮): 

onmousedown 

onmouseup 

onclick 

与 onmouseup 事件相关的事件触发顺序(右边鼠标按钮): 

onmousedown 

onmouseup 

oncontextmenu 

注意： onmouseup 属性不能使用于以下元素: <base>, <bdo>, <br>, 
<head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, 或<title>. 

HTML 4.01 与 HTML5之间的差异 

无。 

语法 

< element onmouseup=" script ">

值 

描述 

script
