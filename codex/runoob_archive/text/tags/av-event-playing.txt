定义和用法 

playing 事件在音频/视频(audio/video)因缓冲而暂停或停止后已就绪时触发。

语法 

HTML 中： 

< audio|video onplaying=" myScript "> 尝试一下 

JavaScript 中： 

audio|video .onplaying=function(){ myScript }; 尝试一下 

JavaScript 中, 使用 addEventListener() 方法: 

audio|video .addEventListener("playing", myScript ); 尝试一下 

注意： Internet Explorer 8 及更早 IE 版本不支持 addEventListener() 方法。 

技术细节
