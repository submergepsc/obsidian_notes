CSS3 :checked 选择器 

实例 

为所有选中的输入元素设置高度与宽度： 

input :checked { height: 50 px ; width: 50 px ; }

定义和用法 

:checked 选择器匹配每个选中的输入元素（仅适用于单选按钮或复选框）。 

:checked 是 CSS3 中的一个伪类选择器，用于选择表单元素中被选中的复选框或单选按钮。当用户点击复选框或单选按钮并选中时，:checked 选择器就会生效。这允许你在用户选择某个选项时改变元素的样式或者进行其他的操作。 

例如，你可以使用 :checked 选择器来改变被选中的复选框或单选按钮的样式，或者将其与其他元素进行联动。这对于创建交互式表单或自定义复选框和单选按钮的外观非常有用。
