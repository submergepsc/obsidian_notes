CSS :optional 选择器 

实例 

如果 input 元素没有设置 "required" 属性，设置其为黄色: 

input:optional 
{ 
background-color: yellow; 
}

定义和使用 

:optional 选择器在表单元素是可选项时设置指定样式。 

表单元素中如果没有特别设置 required 属性即为 optional 属性。 

注意: :optional 选择器只适用于表单元素: input, select 和 textarea。
