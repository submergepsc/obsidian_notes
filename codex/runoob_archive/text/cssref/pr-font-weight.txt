CSS font-weight 属性 

实例 

三段文字设置不同的字体粗细： 

p .normal { font-weight: normal ; } p .thick { font-weight: bold ; } p .thicker { font-weight: 900 ; } 

尝试一下 » 

属性定义及使用说明 

font-weight 属性设置文本的粗细 

默认值： 

normal 

继承： 

yes 

版本： 

CSS1 

JavaScript 语法： 

object .style.fontWeight="900"

默认值。定义标准的字符。

定义由细到粗的字符。400 等同于 normal，而 700 等同于 bold。 

inherit 

规定应该从父元素继承字体的粗细。
