CSS :invalid 选择器 

实例 

如果 input 元素中的值是非法的，设置样式为红色： 

input :invalid { border: 2 px solid red ; } 

尝试一下 » 

定义和使用 

:invalid 选择器用于在表单元素中的值是非法时设置指定样式。 

注意： :invalid 选择器只作用于能指定区间值的元素，例如 input 元素中的 min 和 max 属性，及正确的 email 字段, 合法的数字字段等。
