CSS list-style-image 属性 

实例 

指定列表中的列表项标记的图像： 

ul 
{ 
list-style-image:url('sqpurple.gif'); 
} 

尝试一下 » 

属性定义及使用说明 

list-style-image 属性使用图像来替换列表项的标记。 

注意: 请始终规定一个 "list-style-type" 属性以防图像不可用。 

默认值： 

none 

继承： 

yes 

版本： 

CSS1 

JavaScript 语法： 

object .style.listStyleImage="url('/images/blueball.gif')"
