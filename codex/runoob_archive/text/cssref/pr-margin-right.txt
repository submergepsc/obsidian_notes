属性定义及使用说明 

margin-right属性设置元素的右边距。 

注意： 负值是允许的。 

默认值： 

0 

继承： 

no 

版本： 

CSS1 

JavaScript 语法： 

object .style.marginRight="10px" 

浏览器支持 

表格中的数字表示支持该属性的第一个浏览器版本号。

属性值 

值 

描述 

auto 

浏览器设置的右外边距。 

length 

定义固定的右外边距。默认值是 0。 

% 

定义基于父对象总宽度的百分比右外边距。 

inherit 

规定应该从父元素继承右外边距。
