SVG 参考手册 

SVG 规范中定义了更多的元素和属性。 

常见 SVG 属性： 

id : 元素的唯一标识符。 

class : 元素的类名，用于CSS样式。 

style : 元素的内联样式。 

transform : 应用到元素上的变换。 

x , y : 元素的位置。 

width , height : 元素的尺寸。 

fill : 填充颜色。 

stroke : 描边颜色。 

stroke-width : 描边宽度。 

opacity : 透明度。 

visibility : 元素的可见性。
