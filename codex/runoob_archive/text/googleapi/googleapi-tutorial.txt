什么是 API? 

API = 应用程序编程接口（Application programming interface）。 

API（Application Programming Interface，应用编程接口） 其实就是操作系统留给应用程序的一个调用接口，应用程序通过调用操作系统的 API 而使操作系统去执行应用程序的命令（动作）。
