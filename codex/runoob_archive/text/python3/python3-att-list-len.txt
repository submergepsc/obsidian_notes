Python3 List len()方法 

描述 

len() 方法返回列表元素个数。 

语法 

len()方法语法： 

len(list) 

参数 

list -- 要计算元素个数的列表。 

返回值 

返回列表元素个数。 

实例 

以下实例展示了 len()函数的使用方法： 

实例 

#!/usr/bin/python3 

list1 = [ 'Google' , 'Runoob' , 'Taobao' ] 

print ( len ( list1 ) ) 

list2 = list ( range ( 5 ) ) # 创建一个 0-4 的列表 

print ( len ( list2 ) ) 

以上实例输出结果如下： 

3
5
