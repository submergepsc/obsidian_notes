Python chr() 函数 

描述 

chr() 用一个范围在 range（256）内的（就是0～255）整数作参数，返回一个对应的字符。 

语法 

以下是 chr() 方法的语法: 

chr(i) 

参数 

i -- 可以是10进制也可以是16进制的形式的数字。 

返回值 

返回值是当前整数对应的 ASCII 字符。 

实例 

以下展示了使用 chr() 方法的实例： 

>>> print chr ( 0 x30 ) , chr ( 0 x31 ) , chr ( 0 x61 ) 0 1 a >>> print chr ( 48 ) , chr ( 49 ) , chr ( 97 ) 0 1 a
