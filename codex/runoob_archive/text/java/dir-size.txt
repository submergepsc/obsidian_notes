Java 实例 - 获取目录大小

Main.java 文件 

import java . io . File ; import org . apache . commons . io . FileUtils ; public class Main { public static void main ( String [ ] args ) { long size = FileUtils . sizeOfDirectory ( new File ( " C:/test " ) ) ; System . out . println ( " Size: " + size + " bytes " ) ; } } 

以上代码运行输出结果为： 

Size: 2048 bytes
