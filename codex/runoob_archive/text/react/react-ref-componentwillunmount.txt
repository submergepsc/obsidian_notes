React componentWillUnmount() 方法 

componentWillUnmount() 方法格式如下： 

componentWillUnmount() 

componentWillUnmount() 方法在组件卸载及销毁之前直接调用。 

componentWillUnmount() 方法中不应调用 setState()，因为该组件将永远不会重新渲染。组件实例卸载后，将永远不会再挂载它。
