# wiztree
,另外，listary，和flow launcher
确实，你最近还提到了 **WizTree**。
- **WizTree**: 这是一款**速度极快**的 Windows 磁盘空间分析工具。
  - **核心特点**: 它通过直接读取硬盘的 **MFT (主文件表)** 来扫描文件，所以速度比传统的 WinDirStat 快非常多（几乎是秒级扫描）。
  - **用途**: 通常用来快速查找占用 C 盘或其他硬盘空间的大文件。
