# 三个强大的搜索
### 1. Flow Launcher (开源社区的最佳选择)
这是您之前截图中博主重点推荐的软件，也是目前 Wox 最好的精神续作。
- **定位：** 极速、轻量、完全免费的开源启动器。
- **核心亮点 (Pros):**
  - **社区驱动：** 拥有目前最活跃的开源插件社区，所有插件免费，涵盖了从控制 Spotify 到管理 Docker 容器的各种极客需求。
  - **速度优先：** 代码优化极佳，配合 "Everything" 使用时，文件搜索几乎是瞬时的。
  - **高度定制：** 支持更换各种皮肤，且因为是开源的，没有任何广告干扰。
- **明显不足 (Cons):**
  - **配置门槛：** 部分插件需要系统安装 Python 或 Node.js 环境才能运行，对非技术用户不太友好。
### 2. uTools (功能最全的“瑞士军刀”)
国产效率工具的巅峰之作，它的理念不是“启动应用”，而是“解决问题”。
- **定位：** 本地化的生产力工具箱，插件生态极度丰富。
- **核心亮点 (Pros):**
  - **超级面板：** 独创功能，长按右键或中键呼出面板，能根据你选中的内容（图片/文字）自动推荐功能（如OCR、翻译、上传图床）。
  - **中文生态：** 拥有大量针对国内开发者的插件，如内网穿透、Linux 命令查询、斗图搜索等。
- **明显不足 (Cons):**
  - **商业化限制：** 免费版功能逐渐收缩，多设备同步和部分核心插件需要付费订阅会员。
  - **体积臃肿：** 加载大量插件后，内存占用明显高于 Flow Launcher。
### 3. Listary (文件管理重度依赖者必选)
与其说它是启动器，不如说它是 Windows 文件资源管理器的“外挂”。
- **定位：** 专注于文件搜索与目录跳转的辅助工具。
- **核心亮点 (Pros):**
  - **Ctrl+G 快速跳转：** 这是它的杀手锏。当你在“另存为”窗口时，按下 Ctrl+G 可以立刻跳转到你当前正在浏览的文件夹路径，极大提升存文件的效率。
  - **无感搜索：** 不需要呼出搜索框，直接在桌面或文件夹空白处打字就能搜索文件。
- **明显不足 (Cons):**
  - **插件匮乏：** 虽然也增加了插件功能，但数量和质量远不如 Flow 和 uTools，几乎只能用来找文件。
# 使用listary或者flow launcher，查找目录后使用某个软件快速打开
使用 **Listary** 和 **Flow Launcher** 配合 Typora，主要有两种需求场景：
1. **动态场景：** 搜索到一个任意文件夹，想立刻用 Typora 打开它（作为项目/Vault）。
2. **固定场景：** 设置一个快捷指令（比如输入 `note`），直接打开你最常用的那个笔记文件夹。
前提条件：你需要知道 Typora 的安装路径（通常是 `C:\Program Files\Typora\Typora.exe`）。
------
### 工具一：Listary (推荐，操作最顺滑)
Listary 的优势在于**“动作 (Actions)”**功能，它可以让你在搜索到文件夹后，按一下右方向键调用 Typora。
#### 场景 A：搜索任意文件夹并用 Typora 打开
**设置步骤：**
1. 右键 Listary 图标 -> **选项 (Options/Settings)**。
2. 在左侧菜单选择 **动作 (Actions)**。
3. 点击右侧的 **添加 (Add)** 按钮（通常是个加号）。
4. 填写如下信息：
   - **名称 (Title)**: `Open with Typora` (或者中文“用 Typora 打开”)
   - **路径 (Path)**: 浏览并选择你的 `Typora.exe` 文件。
   - **参数 (Parameter)**: `"{action_path}"`
     - *注意：一定要带引号，防止路径中有空格。Listary 6 可能显示为 `{path}`，请根据界面提示占位符填写，通常下方会有提示，意思是“当前选中的文件路径”。*
5. **保存**。
**如何使用：**
1. 激活 Listary，输入文件夹名字（例如 `myproject`）。
2. 当光标选中该文件夹时，**不要按回车**，而是按键盘上的 **`→` (右方向键)** 或 `Ctrl + O`。
3. 在弹出的动作列表中选择 **Open with Typora**，回车。
------
### 工具二：Flow Launcher
Flow Launcher 逻辑更偏向于“执行命令”或“系统上下文”。
#### 场景 A：搜索任意文件夹并用 Typora 打开 (通过右键菜单)
Flow Launcher 的核心插件 "Explorer" 通常会调用系统的右键菜单。
设置步骤：
不需要在 Flow Launcher 里特别设置，但需要确保你的 Windows 右键菜单里有 "Open with Typora"。
- *如果不确定：* 打开 Typora -> 偏好设置 -> 通用 -> 勾选 **“添加 Typora 到右键菜单”** (Add Typora to context menu)。
**如何使用：**
1. 激活 Flow Launcher，搜索文件夹名。
2. 选中文件夹后，按下 **`Shift + Enter`** (这是呼出上下文菜单的通用快捷键)。
3. 在列表中选择 **Open with Typora** (如果没出现在第一层，可能需要选 "Open With..." 再找)。
#### 场景 B：设置快捷指令打开特定文件夹 (最快，无需搜索文件夹)
如果你每天都要打开同一个“笔记库”，用这个方法最快。使用 **Shell** 插件。
**设置步骤：**
1. 打开 Flow Launcher 设置 -> **Plugins (插件)** -> 找到 **Shell** 或 **Command Line**。
2. 点击 **Add (添加)** 新指令。
3. 填写信息：
   - **Title (标题)**: 打开笔记
   - **Command (命令)**: `"C:\Program Files\Typora\Typora.exe" "D:\我的文档\ObsidianVault"`
     - *格式说明：`"Typora路径" "你的文件夹路径"`。中间有空格，路径都要加英文引号。*
   - **Trigger Keyword (触发关键词)**: `note` (或者你喜欢的任何词)
4. 保存。
**如何使用：**
1. 激活 Flow Launcher。
2. 输入 `note` 并回车。
3. Typora 会直接弹出并加载那个文件夹。
------
### 总结
- **Listary** 适合**灵活操作**：搜到什么文件夹，临时起意想用 Typora 看，直接 `→` 键调用动作。
- **Flow Launcher** 适合**固定入口**：通过 Shell 插件把常用的几个文件夹绑定成关键词（如 `work`、`diary`），一键直达。
# 使用强大的flow launcher
**Flow Launcher** 是 Windows 上最强的**“生产力启动器”**（App Launcher）之一。
如果说 **Listary** 是为了让你在“找文件”这件事上快到极致，那么 **Flow Launcher** 就是为了让你**把手一直放在键盘上**，控制电脑的一切。
它基于 Wox 开发（青出于蓝），开源、免费、速度极快。你可以把它理解为 Windows 版的 **Alfred** 或 **Spotlight**（Mac 上的神级工具）。
------
### 1. 它的核心逻辑：一切皆“关键词”
Flow Launcher 的使用逻辑是：**`触发词` + `空格` + `内容`**。
它不像 Listary 那样只要打字就开始搜文件（虽然它也能搜文件），它更多是用来**“调用功能”**的。
- **计算器**：直接输入 `1+1`，它显示 `2`。
- **翻译**：输入 `tr hello`，它显示 `你好`（需要安装翻译插件）。
- **搜索设置**：输入 `set`，直接跳出 Windows 设置选项。
- **网页搜索**：输入 `g vera`，直接打开 Google 搜索 "vera"。
### 2. 为什么它比 Listary 强大？（插件系统）
Listary 的功能是“定死”的，但 Flow Launcher 有一个庞大的 **插件商店 (Plugin Store)**。这是它最迷人的地方。
你可以通过安装插件实现无数功能：
- **颜色转换**：输入 `#FF0000` 它可以展示颜色预览。
- **汇率转换**：输入 `100 usd` 直接显示人民币金额。
- **天气**：输入 `weather` 看天气。
- **Kill Process**：输入 `kill` 然后选 `Typora`，直接杀掉卡死的进程。
- **Spotify/网易云控制**：切歌、暂停。
### 3. 回到你的问题：为什么你找不到“添加命令”的地方？
这就涉及到 Flow Launcher 的一个核心概念区别：
- **Listary**：把所有功能都做在一个界面里，简单粗暴。
- **Flow Launcher**：**功能都在不同的插件里**。
**你看的那张截图（Command Line / 命令行）**，它的作用是**“代替 Win+R”**。
- 它的功能是：你在框里输入 `ping 8.8.8.8`，它帮你运行这个命令。它**不是**用来存快捷指令的。
你需要的是“Shell”插件（或者叫 CMD Runner）：
正如我刚才提到的，你需要去插件商店（Plugin Store）下载一个专门用来**“保存常用命令”**的插件。那个插件里，才会有 Add（添加）按钮，让你把 note 映射成 打开 Typora。
### 4. 总结：Listary vs Flow Launcher 怎么选？
对于像你这样学计算机的学生，最完美的方案是 **“我全都要”**：
- **Listary**：**专门负责文件操作**。
  - 用它快速跳转文件夹（Ctrl+G）。
  - 用它快速搜索某个深层目录下的代码文件。
  - *把它当成资源管理器的“增强外挂”。*
- **Flow Launcher**：**负责除此之外的一切**。
  - 用来打开软件（输入 `wechat` 启动微信）。
  - 用来做简单计算、翻译。
  - 用来执行你刚才想做的“快捷指令”（输入 `note` 打开笔记库）。
  - *把它当成你的“中央控制台”。*
建议：
现在点击 Flow Launcher 设置里的 Plugin Store（插件商店），在右上角搜索 "Shell" 并安装。装好后，你就能看到那个传说中的 + 号了！
# 几个强大的外村占用查询
想要详细了解C盘哪些内容占据了空间，通常有三种主流方法。你可以根据自己对“详细程度”的需求选择：
### 1. 系统自带方法：存储设置（最安全、无需下载）
这是Windows内置的分析工具，能按类别（应用、临时文件、系统文件）给出直观的统计。
- **操作步骤：**
  1. 按下键盘上的 `Win + I` 打开 **设置**。
  2. 依次点击 **系统** -> **存储**。
  3. 点击 **C盘（本地磁盘）**。
  4. 系统会扫描并显示：**已安装的应用、临时文件、系统和保留、文档**等占用的具体大小。
  5. 点击具体的分类（如“临时文件”），你可以直接勾选并清理不需要的内容（如旧的Windows更新文件、回收站、缩略图等）。
### 2. 专业神器：WizTree 或 TreeSize Free（最详细、强烈推荐）
如果你想知道**具体是哪个文件夹、甚至哪个文件**最大，系统自带的工具就不够用了。推荐使用这两款免费软件，它们会以可视化地图的方式展示磁盘占用：
- **WizTree (推荐)：** 扫描速度极快（几秒钟扫完整个C盘），它会用大大小小的方块展示文件，**方块越大代表文件越大**。你可以一眼看到是哪个游戏的缓存或者哪个视频文件占了空间。
- **TreeSize Free：** 界面类似文件夹管理器，它会按文件夹大小进行排序，你可以像剥洋葱一样一层层点进去，找到那个“罪魁祸首”。
### 3. 查看隐藏的“空间黑洞”
有些占空间的大户是隐藏的，建议手动核实以下几处：
- **用户文件夹 (`C:\Users\用户名`)：** 检查“下载”、“桌面”、“文档”文件夹，尤其是微信/QQ的聊天记录缓存，通常在 `Documents\WeChat Files` 下。
- **虚拟内存与休眠文件：**
  - `hiberfil.sys`：休眠文件，大小通常等于你的内存容量。如果不使用休眠功能，可以以管理员身份运行命令提示符输入 `powercfg -h off` 关闭。
  - `pagefile.sys`：虚拟内存文件。
- **AppData 文件夹：** 位于 `C:\Users\用户名\AppData`（需开启隐藏文件显示）。这里存放了大量软件的缓存和本地数据。
------

您可以先尝试方法1清理系统建议的垃圾；如果空间还是不够，建议下载 WizTree 扫描一下，通常能发现很多被你遗忘的大文件。

# 自启动设置三种方法
如果你的应用在“任务管理器”或“设置”的启动项列表中找不到，通常是因为该应用没有向系统注册自启动服务。
你可以通过以下几种“强制”方法手动设置自启动：
### 方法一：使用“启动”文件夹（最简单推荐）
Windows 有一个专门存放自启动项目快捷方式的文件夹。只要把程序的快捷方式放进去，系统登录时就会自动运行。
1. 按下键盘上的 **Win + R** 键，打开“运行”对话框。
2. 在输入框中输入 `shell:startup`，然后按回车。这会直接打开当前用户的“启动”文件夹。
3. 找到你想要自启动的应用（可以在桌面或开始菜单找到它）。
4. **右键点击**该应用，选择“复制”，然后在刚才打开的“启动”文件夹窗口里**右键点击**空白处，选择“粘贴快捷方式”。
---
### 方法二：通过“任务计划程序”（适合需要管理员权限的应用）
如果你的应用启动时需要弹出“用户账户控制（UAC）”确认窗口，使用方法一可能会失效。此时建议使用任务计划程序。
1. 点击“开始”菜单，搜索并打开 **“任务计划程序”**。
2. 在右侧操作栏点击 **“创建基本任务...”**。
3. 给任务起个名字（如“我的应用自启动”），点击下一步。
4. 触发器选择 **“当前用户登录时”**。
5. 操作选择 **“启动程序”**。
6. 点击“浏览”，找到并选中该应用的 `.exe` 执行文件。
7. 完成后，在任务列表找到它，右键点击“属性”，在“常规”选项卡勾选 **“使用最高权限运行”**（如果是需要管理员权限的应用）。
---
### 方法三：修改注册表（进阶操作）
如果上述方法都不奏效，可以通过注册表手动添加启动键值。
1. 按下 **Win + R**，输入 `regedit` 并回车。
2. 定位到以下路径：
   `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
3. 在右侧空白处**右键 -> 新建 -> 字符串值**，名字随便取（比如应用名）。
4. 双击这个新建的值，在“数值数据”里填入该应用的**完整文件路径**（例如 `"C:\Program Files\App\test.exe"`），注意路径最好带上英文双引号。

# 三种解压缩软件
针对 Bandizip 经常打开失败的情况，结合 2025 年主流解压缩软件的对比，为你推荐以下几款更加稳健、兼容性更强的选择。
### 1. 7-Zip（最强内核，万能解压）
如果 Bandizip 报错，**7-Zip 是你最后的防线**。它体积极小（约 1.6MB），几乎没有它解不开的压缩包。
- **优点：** 完全免费开源，解压率极高，支持 **RAR, 7z, ISO, VHDX, DMG** 等 200 多种格式。它能处理许多因算法更新或损坏而导致其他软件报错的文件。
- **缺点：** 界面非常“复古”（90年代风格），不支持 Windows 11 的新右键菜单（需要点“显示更多选项”）。
- **适用场景：** 处理疑难杂症、追求极致稳定性。
### 2. NanaZip（Windows 11 用户的首选）
如果你嫌 7-Zip 太丑，**NanaZip** 是它的完美现代版。它是基于 7-Zip 核心开发的开源软件，专门为 Win10/11 优化。
- **优点：** 继承了 7-Zip 的所有解压能力，但加入了**暗黑模式**，并且完美集成在 **Windows 11 的第一级右键菜单**中。
- **下载方式：** 直接在 **Microsoft Store (微软商店)** 搜索即可安装，支持自动更新。
### 3. PeaZip（最全的功能和安全性）
如果你需要比解压缩更多的功能（比如文件加密、批量转换），PeaZip 是极佳的选择。
- **优点：** 支持超过 **200 种格式**，界面比 7-Zip 漂亮很多。它还内置了文件粉碎、哈希值校验等高级安全工具。
- **便携性：** 也有 **Portable（便携版）**，不用安装即可使用。
------
### 为什么 Bandizip 会失败？
Bandizip 失败通常不是因为它不够强，而是因为：
1. **文件损坏：** 如果文件在下载过程中丢失了数据（Truncated data），Bandizip 会直接报错而非强行打开。
2. **分卷缺失：** 如果是 `.z01`, `.z02` 这种分卷包，少了一个都会打不开。
3. **算法版本太新：** 有些 RAR5 或特殊的压缩算法，老版本的 Bandizip 可能无法解析。

### 方案一：压缩包加密（最通用、最稳妥）

- **工具**: **7-Zip** (开源免费) 或 WinRAR。

  4. 点击确定。
- **优点**: 跨平台，谁拿走都需要密码，文件结构完全隐藏。

### 方案二：创建加密“保险箱” (VeraCrypt)
如果你是**隐私安全爱好者**（既然你关注指纹浏览器和隐私），这是目前的行业标准，斯诺登级别。
- **工具**: **VeraCrypt** (开源，TrueCrypt 的继承者)。
- **原理**: 它会在电脑上生成一个**文件**（比如 `my_secret.hc`），这个文件挂载起来就像一个**U盘**。
- **操作步骤**:
  1. 安装 VeraCrypt，创建一个 "Encrypted File Container"。
  2. 设置大小（比如 10GB）和密码。
  3. 在软件里选中这个文件，点击 "Mount"（挂载），输入密码。
  4. 此时“我的电脑”里会多出一个盘符（比如 Z: 盘）。
  5. 把你的 `wxid` 文件夹或隐私文件丢进 Z 盘。
  6. 用完点击 "Dismount"，Z 盘消失，变成一个打不开的乱码文件。
- **优点**: 只要不挂载，没人知道里面是什么；支持实时读写。
- **缺点**: 需要安装专门软件。，***android应用的话需要下载***··```EDS(encrypt data store) lite```应用，

### 3. Windows 原生方案：BitLocker + VHD (无需安装第三方软件)
如果你不想下载任何软件，且使用的是 Windows 专业版以上系统。
- **原理：** 利用 Windows 的磁盘管理创建一个“虚拟硬盘文件(VHDX)”，然后用 Windows 自带的 BitLocker 给这个虚拟硬盘上锁。
- **操作简述：**
  1. 右键“此电脑” -> 管理 -> 磁盘管理 -> 操作 -> 创建 VHD。
  2. 初始化这个新磁盘并新建分区。
  3. 在“我的电脑”里右键这个新盘符 -> 启用 BitLocker。
  4. **使用时：** 双击那个 `.vhdx` 文件，它会挂载成一个盘，点击盘符输入密码即可使用。
  5. **关闭时：** 右键盘符选择“弹出”即可。
- *删除vhd的时候，需要先弹出再删除.vhd文件*
- 先初始化再新建简单卷，就能在资源管理器查看了

### ⚠️ 重要警告：千万不要用的方法
1. **EFS 加密 (右键属性 -> 高级 -> 加密内容)**:
   - **极度危险**：这是 Windows 自带的透明加密。它是绑定你的**当前系统账户证书**的。
   - **后果**：如果你重装了系统，或者换了电脑，而没有提前导出密钥证书，这些变绿色的加密文件将**永远打不开**，神仙也救不了。**强烈不建议新手使用。**
2. **不要加密正在运行的微信文件夹**:
   - 你刚才找到的 `xwechat_files` 是微信实时读写的目录。
   - 如果你用软件强行给这个文件夹加密或加锁，微信启动时会因为读取不到数据库文件而**报错崩溃**，甚至导致数据损坏。
   - **正确做法**: 先把微信关掉，把文件夹**复制/备份**出来，再对那个**备份**进行加密。

# 购买信息查询
作为一个追求“极致性价比”的软工学生（毕竟我们刚才聊了半天怎么用几百块拼出高性能硬盘），找对**“情报源”**比找对**“购买渠道”**更重要。
好的价格往往是“稍纵即逝”的（手慢无）。以下是极客和羊毛党们最常用的“蹲好价”体系，按**时效性**和**专业度**分类：
### 1. 综合类：“御用”情报中心
**什么值得买 (SMZDM) App**
- **地位**：国内羊毛党的“圣经”。所有电商（京东/淘宝/拼多多）的优惠几乎都会第一时间汇总到这里。
- **怎么用才爽**：
  - **不要只刷首页**：首页很多是广告。
  - **关注“绝对值”和“历史低价”标签**：这两个标签通常意味着真的便宜。
  - **必杀技【关注关键词】**：把你刚才种草的 `SN740`、`固态硬盘`、`硬盘盒` 添加到关注列表，并开启**推送通知**。一旦有爆料，手机立刻弹窗，这对于抢那些 Bug 价至关重要。
### 2. 极客/数码类：“捡垃圾”大本营
**酷安 (CoolApk)**
- **地位**：你之前的“2230 方案”这种玩法，大部分都是从酷安流出来的。这里的用户人均“图拉丁”（热衷于低成本组装高性能硬件）。
- **核心板块**：
  - **二手交易区**：很多个人玩家出手的回血物资。
  - **话题 #捡垃圾# / #装机#**：在这里能看到最新的“神车”（比如某款企业级硬盘突然大量流出，价格崩盘）。
  - **评论区**：酷安的评论区比正文更有价值，如果一个产品有坑，评论区会毫不留情地喷出来。
### 3. 硬核社区类：最快的情报源
**NGA 玩家社区 (硬件配置区)**
- **地位**：这里的“老哥”说话很直，而且对价格极度敏感。
- **特点**：如果京东突然出现了一个“漏洞价”或者“神券”，NGA 硬件区通常是全网第一个发帖讨论的。
- **搜索关键词**：`开车`、`作业`、`神价`。
**百度贴吧 (图拉丁吧 / 固态硬盘吧)**
- **地位**：最为混沌但也最真实的底层硬件江湖。
- **用途**：
  - **验货**：想知道某家闲鱼店的 SN740 是不是清零盘（翻新盘）？去吧里搜一下店名，如果有黑历史，吧友会挂出来的。
  - **灵车漂移**：这里经常会有一些极其便宜但风险极大的硬件（比如矿卡、服务器拆机件），适合想挑战极限性价比的学生党。
### 4. 交易战场：实战购买地
**闲鱼 (Xianyu)**
- **地位**：你买 SN740、PM991a 这种拆机盘的**唯一指定场所**。
- **防坑秘籍**：
  - **看信用分**：信用极好优先。
  - **看动态**：点进卖家主页，如果他卖的全是同一个型号的硬盘，这是**商家**（虽然叫个人闲鱼）。如果他卖的东西很杂（旧衣服、旧书、旧硬盘），这是**真·个人卖家**（这种最容易捡漏，价格好商量）。
  - **搜索术语**：搜 `拆机`、`回血`、`箱说全`，通常能屏蔽掉很多职业贩子。
### 5. 防忽悠工具：历史价格查询
**慢慢买 (Manmanbuy) / 购物党**
- **用途**：很多时候电商会“先涨价后降价”忽悠人。
- **操作**：把京东/淘宝的商品链接复制进去，看**历史价格曲线**。
  - 如果现在的价格处于曲线的**谷底** $\rightarrow$ 买。
  - 如果现在的价格比上个月还贵 $\rightarrow$ 等。

# 电脑删图便捷方式
确实，手机相册的优势在于“滑动浏览”和“一键删除”的顺滑感，而 Windows 资源管理器默认的列表视图处理图片非常反人类。
要在电脑上达到甚至超越手机的删图效率，你需要改变策略：**不要用鼠标点，要用键盘流。**
前提：请先用 `cppcryptfs` (DroidFS) 或 `VeraCrypt` 把你的加密盘挂载好，确保电脑里能看到那个盘符（比如 X: 盘）。
以下是三种在电脑上高效删图的方案，按推荐程度排序：
### 方案一：使用“看图神器” + 键盘快捷键 (效率最高)
这是摄影师筛选几千张废片的标准操作。比手机滑着删要快得多。
推荐软件：Honeyview (现在叫 BandiView) 或 FastStone Image Viewer
这两款软件都支持“无边框看图”且加载极快（直接读取缓存，不卡顿）。
**操作流：**
1. 打开软件，把你的加密文件夹拖进去。
2. **左手放在 `Delete` 键上，右手放在 `→` (方向键) 上。**
3. 开始浏览：
   - 这张照片要留？按 `→` (下一张)。
   - 这张照片不要？按 `Delete` (删除) -> `Enter` (确认)。
4. **进阶设置：** 在 Honeyview 的设置里，可以把“删除图片”设为**“不显示确认框”**。
   - **效果：** 你看到废片，敲一下 `Del` 它就瞬间消失并自动跳到下一张。**这种“秒删”的节奏感，比手机上点垃圾桶再点确认要快几倍。**
------
### 方案二：优化 Windows 资源管理器 (无需安装软件)
如果你不想装软件，可以通过调整 Windows 自带的视图来模仿手机相册。
1. **打开文件夹：** 进入你的加密盘 X:。
2. **切换视图：**
   - 按住键盘上的 **`Ctrl`** 键，同时滚动鼠标滚轮**向上**。
   - 滚到最大，图标会变成**“超大图标”**模式，这就很像手机相册的网格视图了。
3. **开启预览窗格：**
   - 按下快捷键 **`Alt` + `P`**。
   - 右边会多出一个大预览框。
4. **删图流：**
   - 单击选中第一张图。
   - 看着右边的预览大图。
   - 不喜欢？直接按键盘 **`Delete`**。
   - 喜欢？按方向键 **`→`** 下一张。
------
### 方案三：使用 Windows 自带的“照片”应用 (最像手机)
Windows 10/11 自带的“照片”应用其实就是一个复刻版的手机相册。
1. **右键打开：** 在加密盘里，右键点击任意一张图片 -> 打开方式 -> **照片**。
2. **浏览模式：** 它会以全屏模式显示图片。
3. **操作：**
   - 鼠标滚轮或键盘左右键切换图片。
   - 看到不想要的，按 `Delete` 键（或点击界面上的垃圾桶图标）。
4. **网格视图：** 点击界面左上角的“所有照片”或者类似“集合”的按钮，它会把当前文件夹里的图铺开成照片墙，你可以按住 `Ctrl` 鼠标多选，然后一次性删除。
### ⚠️ 核心警告 (针对加密盘)
无论你用哪种方法，在电脑上删除加密盘（DroidFS/VeraCrypt）里的文件时，**通常不会进入电脑的“回收站”**。
- **特点：** 它是**永久删除**。
- **后果：** 一旦手抖删错，很难找回（因为文件是加密存储的，普通数据恢复软件扫出来也是乱码）。
- **建议：** 如果你使用方案一的“秒删”模式，务必看准了再按键。或者先把不想要的图**剪切 (Ctrl+X)** 移到电脑桌面的一个“待删除文件夹”里，确认无误后再一次性清空。
# 系统信息查询工具---HWINFO64
**HWiNFO64** 是一款专业级的系统信息查询、诊断和实时监控工具，被公认为 Windows 平台上最权威的硬件检测软件之一。它不仅能提供详尽的硬件参数，还能实时监测电脑的运行状态，甚至被 NASA 用于微处理器测试。
以下是 HWiNFO64 的核心功能和应用场景介绍：
### 1. 核心功能
- **深度的硬件信息查询：** 以树状目录展示电脑所有组件的详细规格。你可以看到 CPU 的步进、主板 BIOS 版本、显存颗粒厂商、甚至屏幕面板的具体型号（如是否为京东方屏）。
- **实时传感器监控（Sensors）：** 这是该软件最强大的部分。它可以监控数百个传感器数据，包括：
  - **温度：** CPU 各核心温度、显卡核心/显存温度、硬盘温度等。
  - **电压与功耗：** 实时查看 CPU 和 GPU 的功耗（瓦特），判断电源是否供电不足。
  - **时钟频率：** 监控硬件是否达到了标称频率，或者是否因为过热而降频（Throttling）。
- **日志记录与报告导出：** 支持将监控数据导出为 XML、CSV 或 HTML 格式，方便进行长期的稳定性分析或对比测试。
- **自定义报警：** 你可以设置触发条件，例如当 CPU 温度超过 90°C 时发出警报声或执行特定程序。
------
### 2. 为什么推荐你使用它？
针对你之前的需求（检查硬盘速度和电脑兼容性），HWiNFO64 是最理想的工具：
1. **检测 PCIe 版本：** 在“Bus”选项下，它可以明确告诉你主板插槽是 **Gen 4** 还是 **Gen 5**，从而判断能否跑满致态 TiPro9000 的速度。
2. **诊断过热：** PCIe 5.0 硬盘发热很大，通过传感面板你可以实时监测该硬盘在负载下是否因为达到高温上限而导致降速。
3. **安全无毒：** 该软件不收集个人信息，且提供绿色免安装版（Portable），不会给系统增加垃圾文件。
### 3. 使用建议
- **初学者：** 启动时勾选 **"Summary-only"**，可以快速看到电脑的主要配置简报。
- **进阶玩家：** 勾选 **"Sensors-only"**，专门用于在打游戏或跑分时监控硬件温度和功耗表现。
