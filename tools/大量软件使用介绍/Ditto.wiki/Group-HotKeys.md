# Define hotkey to open a group directly
1. Define hot key to open Group
2. Use that hot key to copy clips directly into that group
Open the clip properties on a group (right click on the group in the group tree - properties), assign the HotKey you want to use.  By default this hotkey will be active when Ditto has focus, pressing it will navigate to the group.  You can also check the check box "hot key available globally", now that group can be accessed without opening Ditto using that hot key (carefull on global hot keys, only so many available on the system).
If "hot key available globally" is check then that hot key can be used to save clips into that group.  Press your global shortcut key to open ditto to that group twice (within 250 milliseconds) and it will copy the current selected text and save it directly into your group.
