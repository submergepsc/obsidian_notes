## powershell use:
#### powershell 原生索引
1. powershell 中一切皆对象，把文件存入数组，然后通过[编号]来引用
```powershell
$f=ls
```

2. 输入变量名来查看文件列表：
```powershell
$f
```

>[!3. 通过索引打开：]+
>*注意索引从0开始*
>- `rm $f[0]`  # 删除第一个文件
>- Invoke-Item $f[2] # 打开第三个文件
>- $f[1] # 查看第二个文件的属性

4. 强制输出序号
```powershell
for ($i=0; $i -lt $f.Count; $i++) { Write-Host "[$i] $($f[$i].Name)" }
```

#### 进阶，创建脚本
1. `win+R`输入notepad, 文件末尾加入以下
```powershell
function lsn {
    $script:f = Get-ChildItem
    for ($i=0; $i -lt $script:f.Count; $i++) {
        Write-Host ("[{0,2}] {1}" -f $i, $script:f[$i].Name) -ForegroundColor Cyan
    }
}
```

2. 多个文件删除
```powershell
rm $f[25...30] #删除连续多个
rm $f[25,28,31] #不连续
rm $f[(1...5)+8+10] #组合
```
