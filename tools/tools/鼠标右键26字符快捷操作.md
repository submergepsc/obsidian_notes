在 Windows 操作系统中，右键菜单（上下文菜单）的字母快捷键是一项极大地提升“极客”效率的隐藏功能。当你按下右键（或键盘上的 Menu 键）后，直接点击对应的字母，即可触发该选项。
需要注意的是，这些快捷键的分配逻辑分为两类：**系统硬编码快捷键**（如 `M` 总是重命名）和**动态分配快捷键**（菜单项名称后括号里的字母，如 `(N)`）。
---
## 一、 文件/文件夹上的右键快捷键
这是最常用的场景，以下是标准 Windows 11/10 菜单（旧版布局或 Shift+右键）中的常用映射：

| **字母** | **功能 (中文环境)**     | **常用场景**            |
| ------ | ----------------- | ------------------- |
| **O**  | 打开 (Open)         | 替代双击                |
| **S**  | 创建快捷方式 (Shortcut) | 在当前目录生成快捷图标         |
| **C**  | 复制 (Copy)         | 配合 Ctrl+V 使用        |
| **T**  | 剪切 (Cut)          | 移动文件                |
| **V**  | 粘贴 (Paste)        | 仅在文件夹空白处或文件夹图标上有效   |
| **D**  | 删除 (Delete)       | 移至回收站               |
| **M**  | 重命名 (Rename)      | 快速修改文件名（比 F2 有时更顺手） |
| **R**  | 属性 (Properties)   | 查看文件大小、权限           |
| **N**  | 新建 (New)          | 通常在文件夹空白处呼出二级菜单     |
| **H**  | 共享 (Share)        | 唤起系统分享面板            |
| **A**  | 全选 (Select All)   | 在某些特定容器插件中有效        |
| **E**  | 编辑 (Edit)         | 针对图片（画图）或脚本文件       |
| **P**  | 打印 (Print)        | 针对文档或图片             |
| **G**  | 提取全部 (Extract)    | 针对 .zip 压缩包         |
| **W**  | 包含到库中             | 管理资源管理器库            |
| **X**  | 卸载                | 针对虚拟光驱挂载的镜像         |
---
## 二、 桌面或文件夹空白处的快捷键
在没有选中文件的情况下，右键点击空白处：
- **V**: 查看 (View) - 随后可按 **L** (大图标)、**M** (中图标) 等。
- **O**: 排序方式 (Sort by) - 随后可按 **N** (名称)、**S** (大小)。
- **E**: 刷新 (Refresh) - 经典的除错/手动同步动作。
- **W**: 新建 (New) - 随后可按 **F** (文件夹)、**T** (文本文档)。
- **R**: 屏幕设置 (Display settings)。
- **P**: 个性化 (Personalize)。
---
## 三、 第三方软件的扩展快捷键
许多你安装的工具会抢占剩余的字母位：
- **B**: 使用 百度网盘/某些备份工具上传。
- **I**: 使用 7-Zip 或 WinRAR 压缩/解压（取决于版本）。
- **K**: 使用 某些杀毒软件扫描 (Scan)。
- **Z**: 某些云盘的同步选项。

## 四、 进阶技巧与注意事项
### 1. 字母冲突怎么办？
如果菜单中有两个选项都带有 `(A)`，按下 `A` 键时，系统不会立即执行，而是会在两个选项之间**循环高亮**。此时需要按下 **Enter** 键来确认。
### 2. Windows 11 的“双层菜单”
Windows 11 默认的简化菜单不直接显示这些字母。
- **解决方法**：按下右键后，按一下键盘上的 `Shift + F10`（或开启“始终显示旧版菜单”），字母快捷键才会完全激活。
### 3. 如何识别快捷键？
观察右键菜单，选项文字后面带有下划线的字母，或者括号里的字母，就是该项的快捷键。
> 例如：发送到 (N) -> 此时按 **N** 即可。
### 4. 推荐的“极客组合拳”
- **右键 + W + F**：极速新建文件夹。
- **右键 + W + T**：极速新建记事本。
- **右键 + M**：文件重命名。
- **右键 + R**：查看文件属性（比 Alt+Enter 适合单手操作）。
