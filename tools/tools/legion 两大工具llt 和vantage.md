### 1. Lenovo Vantage (官方管家)
作为联想出厂预装的控制中心，它的定位是“大而全”，适合完全不想折腾的普通用户。
- **优势**：
    - **开箱即用**：买来就有，UI 动画华丽。
    - **驱动与 BIOS 更新**：集成了官方的驱动推送通道（虽然有时推送较慢）。
    - **保修与客服服务**：内置了直接查询机器保修期和联系客服的入口。
- **劣势 (痛点)**：
    - **极度臃肿**：它由几十个后台服务（如 `ImControllerService`）和进程组成，即使你不打开界面，后台也会常驻占用几十甚至上百 MB 的内存，并持续消耗 CPU 资源。
    - **广告与遥测**：包含大量的捆绑推广（如推荐杀毒软件、游戏加速器）以及用户数据遥测收集。
    - **功能受限**：风扇控制和性能调度只给最基础的预设，无法进行极客级别的微调。
- **官方下载链接**：[Microsoft Store - Lenovo Vantage](https://apps.microsoft.com/detail/9wzdncrfj4mv)
### 2. Lenovo Legion Toolkit (LLT)
这是一款由开发者在 GitHub 上开源的轻量级替代工具，通过调用联想官方的底层接口（WMI 和 ACPI）来实现控制。
- **优势**：
    - **极致轻量**：安装包极小，运行时内存占用通常在 20MB 以内。最关键的是，**关闭界面后它没有任何后台服务常驻**（除非你开启了自动化规则），真正做到零额外负载。
    - **零广告、开源**：代码完全公开透明，没有任何商业推广和数据收集。
    - **高级自定义与自动化**：不仅能精细绘制双风扇曲线，还能设定“操作 (Actions)”。例如：拔掉电源 -> 自动切换 60Hz 刷新率 -> 关闭键盘背光 -> 切换安静模式；插上电源 -> 恢复 165Hz -> 开启野兽模式。
    - **显卡模式无缝切换**：在支持的机型上（如 Y7000P），切换混合输出/直连模式更加迅速，不需要经过繁琐的官方 UI 动画。
- **劣势**：
    - **不包含驱动更新**：你需要自己通过 Windows Update 或 NVIDIA 官方渠道更新驱动（这通常也是极客更推荐的更新方式）。
- **官方下载链接**：[GitHub Releases - Lenovo Legion Toolkit](https://github.com/BartoszCichecki/LenovoLegionToolkit/releases)
