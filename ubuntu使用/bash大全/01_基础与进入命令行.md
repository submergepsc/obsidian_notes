# 基础与进入命令行

## Bash 是什么

Bash 是一种 Shell。

它的作用：

- 接收你输入的命令
- 解释命令语法
- 调用系统程序执行
- 支持脚本自动化

## 最基础命令

### `pwd`

显示当前所在目录。

```bash
pwd
```

### `cd`

切换目录。

```bash
cd /home
cd ..
cd ~
cd -
```

常见形式：

- `cd ..`：上一级
- `cd ~`：家目录
- `cd -`：上一次所在目录

### `ls`

列出目录内容。

```bash
ls
ls -l
ls -a
ls -la
```

常见参数：

- `-l`：详细信息
- `-a`：包含隐藏文件

### `clear`

清屏。

```bash
clear
```

### `history`

查看历史命令。

```bash
history
history | tail
```

### `exit`

退出当前 shell。

```bash
exit
```

## 帮助命令

### `help`

查看 Bash 内建命令帮助。

```bash
help cd
help read
```

### `man`

查看系统命令手册。

```bash
man ls
man grep
```

### `--help`

快速查看命令选项。

```bash
ls --help
grep --help
```

### `type`

判断命令是什么。

```bash
type cd
type ls
type echo
```

### `which`

查看外部命令路径。

```bash
which bash
which python3
```

## 自动补全与快捷键

- `Tab`：命令或路径补全
- `Ctrl + C`：中断当前命令
- `Ctrl + D`：结束输入或退出 shell
- `Ctrl + L`：清屏，类似 `clear`
- `Ctrl + A`：移动到行首
- `Ctrl + E`：移动到行尾
- `Ctrl + R`：搜索历史命令

## alias

给命令起别名。

```bash
alias ll='ls -la'
alias gs='git status'
```

取消别名：

```bash
unalias ll
```
