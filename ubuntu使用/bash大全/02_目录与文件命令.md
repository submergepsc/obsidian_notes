# 目录与文件命令

## 目录操作

### `mkdir`

创建目录。

```bash
mkdir test
mkdir -p a/b/c
```

### `rmdir`

删除空目录。

```bash
rmdir empty_dir
```

### `tree`

树状显示目录结构。

```bash
tree
tree -L 2
```

如果系统没有安装：

```bash
sudo apt install tree
```

## 文件查看与创建

### `touch`

创建空文件，或更新时间戳。

```bash
touch a.txt
```

### `file`

判断文件类型。

```bash
file a.txt
file image.png
```

### `stat`

查看文件详细状态。

```bash
stat a.txt
```

## 复制、移动、删除

### `cp`

复制文件或目录。

```bash
cp a.txt b.txt
cp -r dir1 dir2
```

### `mv`

移动或重命名。

```bash
mv old.txt new.txt
mv file.txt ./backup/
```

### `rm`

删除文件或目录。

```bash
rm a.txt
rm -r dir
rm -rf dir
```

注意：

- `rm -rf` 很危险
- 删除后通常不能轻易恢复

## 链接

### `ln`

创建链接。

```bash
ln source.txt hardlink.txt
ln -s /path/to/source softlink
```

说明：

- 硬链接：共享 inode
- 软链接：类似快捷方式

## 文件权限相关

### `chmod`

修改权限。

```bash
chmod +x script.sh
chmod 755 script.sh
```

### `chown`

修改所有者。

```bash
sudo chown user:user file.txt
```

### `chgrp`

修改所属组。

```bash
chgrp developers file.txt
```

## 压缩与归档

### `tar`

打包、解包。

```bash
tar -czf backup.tar.gz mydir
tar -xzf backup.tar.gz
```

### `zip` / `unzip`

压缩与解压 zip 文件。

```bash
zip -r project.zip project
unzip project.zip
```

## 查找文件

### `find`

按名称、类型、时间等查找。

```bash
find . -name "*.md"
find . -type d
find . -size +100M
```

### `locate`

基于数据库快速查找。

```bash
locate bashrc
```

第一次可能要更新数据库：

```bash
sudo updatedb
```
