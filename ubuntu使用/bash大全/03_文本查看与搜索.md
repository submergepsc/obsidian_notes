# 文本查看与搜索

## 查看文件内容

### `cat`

直接输出文件全部内容。

```bash
cat file.txt
cat a.txt b.txt
```

### `less`

分页查看，适合长文件。

```bash
less file.txt
```

常见按键：

- `q`：退出
- `/关键字`：搜索
- `n`：下一个匹配

### `more`

简单分页查看。

```bash
more file.txt
```

### `head`

看开头几行。

```bash
head file.txt
head -n 20 file.txt
```

### `tail`

看末尾几行。

```bash
tail file.txt
tail -n 50 file.txt
tail -f app.log
```

## 文本搜索

### `grep`

按关键字匹配文本。

```bash
grep "error" app.log
grep -n "main" file.c
grep -r "TODO" .
grep -i "bash" note.md
```

常见参数：

- `-n`：显示行号
- `-r`：递归搜索
- `-i`：忽略大小写

### `egrep`

等价于 `grep -E`，支持扩展正则。

```bash
grep -E "cat|dog" animals.txt
```

### `fgrep`

等价于 `grep -F`，按普通字符串搜索。

```bash
grep -F "[abc]" file.txt
```

## 文本处理

### `sort`

排序。

```bash
sort names.txt
sort -n scores.txt
sort -r names.txt
```

### `uniq`

去重，通常与排序联用。

```bash
sort names.txt | uniq
sort names.txt | uniq -c
```

### `wc`

统计行数、单词数、字节数。

```bash
wc file.txt
wc -l file.txt
```

### `cut`

按列截取文本。

```bash
cut -d: -f1 /etc/passwd
cut -c1-10 file.txt
```

### `paste`

按列合并文件。

```bash
paste a.txt b.txt
```

### `tr`

替换或删除字符。

```bash
echo "abc" | tr 'a-z' 'A-Z'
echo "a b c" | tr -d ' '
```

### `sed`

流编辑器，擅长替换、选行。

```bash
sed 's/old/new/' file.txt
sed -n '1,10p' file.txt
```

### `awk`

按列处理文本，非常强大。

```bash
awk '{print $1}' file.txt
awk -F: '{print $1}' /etc/passwd
```

## 差异比较

### `diff`

比较两个文件差异。

```bash
diff a.txt b.txt
diff -u a.txt b.txt
```

### `cmp`

按字节比较文件。

```bash
cmp a.bin b.bin
```
