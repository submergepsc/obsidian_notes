# 流程控制 函数 脚本

## 条件判断

### `test`

测试条件。

```bash
test -f file.txt
```

### `[ ]`

最常见判断写法。

```bash
[ -f file.txt ]
[ "$a" = "$b" ]
[ "$n" -gt 10 ]
```

### `[[ ]]`

Bash 更强的判断语法，推荐在 Bash 脚本里使用。

```bash
[[ -f file.txt ]]
[[ "$name" == *.md ]]
```

## `if`

```bash
if [[ -f file.txt ]]; then
  echo "存在"
else
  echo "不存在"
fi
```

## `case`

适合多分支匹配。

```bash
case "$1" in
  start) echo "启动" ;;
  stop) echo "停止" ;;
  *) echo "未知参数" ;;
esac
```

## 循环

### `for`

```bash
for file in *.md; do
  echo "$file"
done
```

### `while`

```bash
while read line; do
  echo "$line"
done < file.txt
```

### `until`

条件不成立时循环。

```bash
count=1
until [ "$count" -gt 3 ]; do
  echo "$count"
  count=$((count + 1))
done
```

## `break` 与 `continue`

```bash
for i in 1 2 3 4 5; do
  if [ "$i" -eq 3 ]; then
    continue
  fi
  echo "$i"
done
```

## 函数

```bash
say_hello() {
  echo "hello $1"
}

say_hello tom
```

## 返回值

Shell 里命令执行后都会有退出状态码。

```bash
echo $?
```

约定：

- `0`：成功
- 非 `0`：失败

## 算术运算

```bash
a=3
b=5
sum=$((a + b))
echo "$sum"
```

## 脚本文件

### 第一行 shebang

```bash
#!/usr/bin/env bash
```

### 执行脚本

```bash
bash test.sh
chmod +x test.sh
./test.sh
```

## 常见特殊变量

- `$0`：脚本名
- `$1` `$2`：第 1、第 2 个参数
- `$#`：参数个数
- `$@`：所有参数
- `$?`：上个命令退出码
- `$$`：当前进程 PID

例子：

```bash
echo "script=$0"
echo "first=$1"
echo "count=$#"
```
