# 进程 任务 权限 网络

## 进程管理

### `ps`

查看进程。

```bash
ps
ps aux
ps -ef
```

### `top`

动态查看系统资源。

```bash
top
```

### `htop`

更友好的进程查看工具。

```bash
htop
```

如未安装：

```bash
sudo apt install htop
```

### `kill`

按 PID 终止进程。

```bash
kill 1234
kill -9 1234
```

### `pkill`

按进程名终止。

```bash
pkill python
```

## 前台后台任务

### `&`

让命令在后台运行。

```bash
python app.py &
```

### `jobs`

查看当前 shell 的后台任务。

```bash
jobs
```

### `bg`

把任务放到后台继续执行。

```bash
bg %1
```

### `fg`

把后台任务拉回前台。

```bash
fg %1
```

### `nohup`

退出终端后仍继续运行。

```bash
nohup python app.py &
```

## 权限与身份

### `sudo`

以管理员权限执行命令。

```bash
sudo apt update
sudo systemctl restart ssh
```

### `su`

切换用户。

```bash
su
su username
```

### `id`

查看用户 UID/GID。

```bash
id
```

### `whoami`

显示当前用户名。

```bash
whoami
```

## 系统信息

### `uname`

查看系统内核信息。

```bash
uname -a
```

### `hostname`

查看主机名。

```bash
hostname
```

### `date`

查看日期时间。

```bash
date
```

### `uptime`

查看运行时长和负载。

```bash
uptime
```

### `df`

查看磁盘空间。

```bash
df -h
```

### `du`

查看目录大小。

```bash
du -sh .
du -sh *
```

### `free`

查看内存。

```bash
free -h
```

## 网络相关

### `ping`

测试连通性。

```bash
ping baidu.com
```

### `curl`

请求 URL，常用于 API 调试和下载。

```bash
curl https://example.com
curl -I https://example.com
```

### `wget`

下载文件。

```bash
wget https://example.com/file.zip
```

### `ssh`

远程登录。

```bash
ssh user@host
```

### `scp`

远程复制文件。

```bash
scp file.txt user@host:/tmp/
scp user@host:/tmp/file.txt .
```

## 软件包管理

### `apt`

Ubuntu 最常见包管理命令。

```bash
sudo apt update
sudo apt upgrade
sudo apt install tree
sudo apt remove tree
```

### `dpkg`

管理 `.deb` 包。

```bash
dpkg -i package.deb
dpkg -l
```
