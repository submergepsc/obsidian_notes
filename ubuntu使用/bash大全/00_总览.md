# Bash大全

## 这套笔记的范围

严格意义上，“bash 的所有命令”无法一次写完，因为：

- Bash 自己有一批内建命令（builtin）
- Linux/Ubuntu 还提供大量外部命令
- 不同软件安装后还会继续新增命令

所以这套目录采用更实用的范围：

- Bash 核心语法
- Bash 内建命令
- Ubuntu 常用命令
- 按场景分类整理，便于查找

## 如何判断一个命令属于哪类

- `cd`、`echo`、`read`、`export`：常见 Bash 内建命令
- `ls`、`cp`、`mv`、`grep`、`find`：常见外部命令
- `if`、`for`、`while`：Shell 语法关键字，不是普通可执行程序

## 自查命令

```bash
type cd
type ls
help cd
man ls
which python3
```

说明：

- `type`：查看命令属于 builtin、alias 还是外部程序
- `help`：查看 Bash 内建命令帮助
- `man`：查看系统手册
- `which`：查看外部程序路径
