# 变量 重定向 管道

## 变量

### 定义变量

```bash
name="loviya"
age=18
```

注意：等号两边不能有空格。

### 使用变量

```bash
echo "$name"
echo "${age}"
```

### 只读变量

```bash
readonly name
```

### 删除变量

```bash
unset age
```

## 环境变量

### `export`

导出环境变量，让子进程可见。

```bash
export JAVA_HOME=/usr/lib/jvm/default-java
export PATH="$PATH:/my/bin"
```

### `env`

查看环境变量。

```bash
env
env | grep PATH
```

### `printenv`

打印环境变量。

```bash
printenv
printenv HOME
```

## 输入输出

### `echo`

输出文本或变量。

```bash
echo hello
echo "$HOME"
echo -e "a\nb"
```

### `printf`

格式化输出。

```bash
printf "name=%s age=%d\n" "tom" 20
```

### `read`

从标准输入读取内容。

```bash
read name
read -p "请输入名字: " name
```

## 重定向

### `>`

覆盖写入文件。

```bash
echo hello > a.txt
```

### `>>`

追加写入文件。

```bash
echo world >> a.txt
```

### `<`

从文件读取输入。

```bash
wc -l < file.txt
```

### `2>`

重定向错误输出。

```bash
ls not_exist 2> error.log
```

### `&>`

标准输出和错误输出一起重定向。

```bash
command &> all.log
```

## 管道

### `|`

把前一个命令输出交给后一个命令处理。

```bash
ls -la | grep ".md"
ps aux | grep python
history | tail
```

## 命令替换

### `$(...)`

把命令执行结果作为值使用。

```bash
today=$(date)
files=$(ls)
```

旧写法也存在，但不推荐：

```bash
today=`date`
```

## 通配符

- `*`：匹配任意多个字符
- `?`：匹配一个字符
- `[abc]`：匹配集合内一个字符

例子：

```bash
ls *.md
ls file?.txt
```

## 引号

### 单引号 `'`

内容原样保留，不展开变量。

```bash
echo '$HOME'
```

### 双引号 `"`

保留整体字符串，但允许变量展开。

```bash
echo "$HOME"
```

### 反斜杠 `\`

转义特殊字符。

```bash
echo \$HOME
```
