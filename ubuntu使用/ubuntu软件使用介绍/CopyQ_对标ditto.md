
明白你的意思了！你需要的不是简单的“记录一下历史”，而是像 **Ditto** 那样级别的**重型效率工具**：能跨软件全局调用、能存海量历史、支持图片和富文本、能秒速搜索、且重启后数据不丢失的剪贴板神器。

前面的 `Clipboard Indicator` 确实太轻量了，顶多算个玩具。在 Linux 生态里，真正能完美对标（甚至在可玩性上超越）Ditto 的，只有这款开源神器：**CopyQ**。

### 终极对标方案：CopyQ

它几乎复刻了 Ditto 的所有核心体验，甚至因为支持自定义脚本，非常契合你这种喜欢深度定制的 Geek 玩家。

#### 1. 极速安装
直接打开终端，用 apt 安装：
```bash
sudo apt update
sudo apt install copyq
```

#### 2. 核心设置（打造 Ditto 体验）
安装完成后，在应用列表里打开 **CopyQ**，它的图标会出现在右上角的托盘区。
1. 右键托盘图标，进入 **Preferences（首选项）**。
2. 在 **History（历史）** 标签页，你可以像 Ditto 一样设置保存的条目数量（比如 1000 条）。
3. 确保勾选了对图片和富文本的支持。

#### 3. 绑定全局快捷键（最关键的一步）
Ditto 的灵魂在于一键呼出搜索框。Ubuntu 的全局快捷键需要这样接管：
1. 打开 Ubuntu 系统的 **设置 (Settings)** -> **键盘 (Keyboard)** -> **查看及自定义快捷键 (View and Customize Shortcuts)**。
2. 划到最底部，点击 **自定义快捷键 (Custom Shortcuts)**，然后添加一个全新的快捷键：
   * **名称：** `呼出 CopyQ`
   * **命令：** `copyq toggle` （这个命令会立刻在鼠标位置弹出主面板）
   * **快捷键：** 按下你习惯的按键（比如 `Super + V` 或者 `Ctrl + \`）。

#### CopyQ 为什么能平替 Ditto？
* **秒速搜索：** 按下快捷键呼出面板后，**直接敲击键盘输入字母**，它就会像 Ditto 一样立刻对历史记录进行全文过滤。
* **分栏与标签：** 支持建立多个 Tab 页（比如把常用的代码片段固定在一个 Tab 里，像本地的代码片段库一样）。
* **无缝粘贴：** 选中记录回车，直接粘贴到当前光标所在的非终端软件中（如浏览器、Obsidian、IDE 等）。
* **纯文本粘贴：** 带有强悍的格式刷功能，可以强制去除网页复制带来的恶心格式。

配置好快捷键后，你日常在任何 GUI 软件里写文档或敲代码，直接一套快捷键呼出面板，敲几个字母回车上屏，手感和在 Windows 用 Ditto 绝对是一模一样的。
