# 🥉 经典原版（系统自带）

## 👉 `top`

```bash
top
```

### 操作：

* `k` → 杀进程
* `P` → 按 CPU 排序
* `M` → 按内存排序

👉 功能强，但界面丑（纯文本）

## 👉 `kill`

```bash

```

👉 强制杀死

# 🔥 给你一个“爽用组合”（推荐）

你现在的使用习惯，我建议：

```bash
htop
```

然后：

* `/` 搜索进程（比如 wechat）
* `F9` 杀掉卡死的 Wine 程序
* `F5` 看进程树（调试特别爽）
