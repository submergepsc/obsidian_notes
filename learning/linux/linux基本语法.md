### 1. **文件操作**
- **创建文件**
  ```bash
  touch 文件名          # 创建空文件
  echo "内容" > 文件名  # 创建文件并写入内容
  ```
- **查看文件内容**
  ```bash
  cat 文件名            # 查看文件内容
  less 文件名           # 分页查看文件内容
  more 文件名           # 分页查看文件内容（较少功能）
  head -n 行数 文件名   # 查看文件前 n 行
  tail -n 行数 文件名   # 查看文件后 n 行
  ```
- **复制、移动、删除文件**
  ```bash
  cp 源文件 目标路径     # 复制文件
  mv 源文件 目标路径     # 移动文件或重命名
  rm 文件名             # 删除文件
  ```

### 2. **目录管理**
- **查看当前目录**
  ```bash
  pwd                   # 显示当前工作目录
  ```
- **切换目录**
  ```bash
  cd 目录路径           # 切换到指定目录
  cd ..                 # 返回上一级目录
  cd ~                  # 切换到用户主目录
  ```
- **创建、删除目录**
  ```bash
  mkdir 目录名          # 创建目录
  mkdir -p 路径         # 递归创建多级目录
  rmdir 目录名          # 删除空目录
  rm -r 目录名          # 递归删除目录及其内容
  ```

### 3. **权限管理**
- **查看权限**
  ```bash
  ls -l                # 显示文件和目录的详细信息，包括权限
  ```
- **修改权限**
  ```bash
  chmod 模式 文件名    # 修改文件权限（如 chmod 755 文件名）
  chmod u+x 文件名     # 给文件的所有者添加执行权限
  chmod g-w 文件名     # 移除组的写权限
  chmod o=r 文件名     # 设置其他用户只读权限
  ```
- **修改所有者和组**
  ```bash
  chown 用户名 文件名  # 修改文件所有者
  chgrp 组名 文件名    # 修改文件所属组
  ```

### 4. **进程管理**
- **查看进程**
  ```bash
  ps                   # 查看当前用户的进程
  ps aux               # 查看所有进程
  top                  # 实时查看系统进程
  htop                 # 更友好的实时进程查看工具（需安装）
  ```
- **管理进程**
  ```bash
  kill PID             # 终止指定 PID 的进程
  killall 进程名        # 终止指定名称的所有进程
  bg                   # 将挂起的进程放到后台运行
  fg                   # 将后台进程调回前台
  ```

### 5. **用户管理**
- **切换用户**
  ```bash
  su 用户名            # 切换到指定用户
  sudo 命令            # 以管理员权限执行命令
  ```
- **用户管理**
  ```bash
  whoami               # 显示当前用户
  who                  # 显示当前登录的所有用户
  adduser 用户名       # 添加用户
  passwd 用户名        # 修改用户密码
  ```

### 6. **网络操作**
- **查看网络状态**
  ```bash
  ifconfig             # 查看网络接口信息
  ip addr              # 查看网络接口信息（推荐）
  ping 地址            # 测试网络连通性
  ```
- **下载文件**
  ```bash
  wget URL             # 下载文件
  curl -O URL          # 下载文件
  ```

### 7. **磁盘管理**
- **查看磁盘使用情况**
  ```bash
  df -h                # 查看磁盘空间使用情况
  du -h 文件或目录      # 查看文件或目录的大小
  ```
- **挂载与卸载**
  ```bash
  mount 设备 目录       # 挂载设备到目录
  umount 目录          # 卸载挂载点
  ```

### 8. **软件管理**
- **基于 Debian 系统（如 Ubuntu）**
  ```bash
  apt update           # 更新软件源
  apt install 软件名    # 安装软件
  apt remove 软件名     # 卸载软件
  ```
- **基于 Red Hat 系统（如 CentOS）**
  ```bash
  yum update           # 更新软件源
  yum install 软件名    # 安装软件
  yum remove 软件名     # 卸载软件
  ```

### 9. **压缩与解压**
- **压缩文件**
  ```bash
  tar -cvf 文件名.tar 文件或目录  # 打包文件
  tar -czvf 文件名.tar.gz 文件或目录  # 打包并压缩
  ```
- **解压文件**
  ```bash
  tar -xvf 文件名.tar             # 解包
  tar -xzvf 文件名.tar.gz         # 解压缩
  unzip 文件名.zip                # 解压 ZIP 文件
  ```

### 10. **其他常用命令**
- **查看系统信息**
  ```bash
  uname -a              # 查看系统内核信息
  lsb_release -a        # 查看操作系统版本
  ```
- **清屏**
  ```bash
  clear                 # 清空终端屏幕
  ```
- **查看命令历史**
  ```bash
  history               # 显示历史命令
  ```
