### 斜杠命令
```
使用/+number,自动生成标题
	并且还有很多其他的命令
```


`ctrl+T`:新标签页
`ctrl+clikc`:新标签页打开
`ctrl+N`:新文件

## 插件使用
### `advanced new file`
> [! 选项介绍]+
> Advanced new file: Create note in a new tab     新标签页
Advanced new file: Create note in the current pane      在当前面板但是替换当前标签页
Advanced new file: Create note in a new pane      在新的面板

### `terminal`
>[!配置简介]+
>就整合式和外部式，但是外部终端都整合式一直删不掉




- **斜杠命令**:
  - 使用 `/+number` 自动生成标题。
  - 提供多种其他命令。

- **快捷键**:
  - `Ctrl+T`: 打开新标签页。
  - `Ctrl+Click`: 在新标签页中打开。
  - `Ctrl+N`: 创建新文件。

- **插件使用**:
  - **Advanced New File**:
    - 创建笔记在新标签页中。
    - 创建笔记在当前面板中（替换当前标签页）。
    - 创建笔记在新面板中。
  - **Terminal**:
    - 配置分为整合式和外部式。
    - 外部终端始终无法删除整合式。
